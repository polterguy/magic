Magic is an automation framework built in ASP.NET Core. Its purpose is to allow you to "magically" do the boring stuff, by using automation techniques. It features among other things an HTTP Web API REST CRUD generator, that will automatically create CRUD endpoints wrapping your MySQL or MS SQL Server database - In addition to an Angular frontend scaffolder, to generate a default frontend for you.

Although most of Magic is Open Source, you do need a license key to run it in production. You can obtain a license here. Or contact us on license@servergardens.com for licensing inquiries.
